# Lessons Learned

This directory stores concise bug and failure lessons for future agents.

Use one file per lesson:

```text
YYYY-MM-DD-short-topic.md
```

Each note should include only:

- Symptom
- Root cause
- Fix
- Prevention

Do not store general architecture decisions here. Put stable project context and decisions in `.cursor/project-context/`.

Keep notes concise and searchable.
